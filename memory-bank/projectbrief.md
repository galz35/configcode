# Project Brief — Configcode v2

## Descripción del Proyecto
`configcode` es un framework de configuración universal, agnóstico de IDE y optimizado para agentes de Inteligencia Artificial (IA) aplicados a la codificación. Su objetivo es convertir cualquier repositorio de software en un entorno ultra-legible e instructivo para que las IAs (como Claude, Gemini, DeepSeek, GPT) puedan codificar con la máxima calidad, seguridad, y consistencia sin requerir explicaciones manuales reiterativas.

## Objetivos Clave
- **Cero ORMs:** Forzar el uso de consultas SQL puras y parametrizadas para garantizar un rendimiento óptimo y evitar abstracciones innecesarias.
- **Portabilidad de Contexto:** Proporcionar un sistema de "Memory Bank" persistente que mantenga la coherencia del estado del proyecto en cada sesión de chat de IA.
- **Alineación Visual y de Estilo:** Definir pautas estrictas de diseño UX premium, HSL tokens, y accesibilidad universal.
- **Automatización de Tareas:** Proveer agentes especializados, comandos cortos, y herramientas automáticas de validación técnica.
