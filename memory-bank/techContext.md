# Tech Context — Configcode v2

## Entorno del Proyecto
`configcode` es un repositorio de configuración estructurado en Markdown, scripts de TypeScript para validaciones, y plantillas de código para andamiaje (scaffolding).

## Stack Técnico
- **Formatos:** Markdown (.md) para reglas, guías de estilo, y especificaciones de agentes/comandos.
- **Herramientas de Validación (Tools):** TypeScript (.ts) utilizando el framework `@opencode-ai/plugin` para ejecutar validaciones estáticas.
- **Plantillas (Templates):**
  - **Express API:** Node.js, Express, TypeScript, `mssql`, `pg`.
  - **React Vite:** React, Vite, TypeScript, Zustand, CSS.
  - **Flutter App:** Dart, Flutter, Riverpod, GoRouter, Hive.

## Restricciones y Pautas
- Los scripts TypeScript de las herramientas deben correr utilizando Node.js sin requerir configuraciones complejas de build en la raíz.
- Toda la documentación debe redactarse en español claro, técnico, y orientado a la practicidad absoluta.
